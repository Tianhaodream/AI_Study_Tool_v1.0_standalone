【添加新功能的步骤】

1. 查看 backend/main.py 中的 API 端点
2. 在 Flask app 中添加新的路由
3. 修改 frontend/index.html 中的 JavaScript 来调用新端点
4. 更新 小白使用说明.md 中的用户文档
5. 测试功能，更新 完整系统验证报告.md
6. 运行 release_all.py 生成新版本

【修复 Bug 的步骤】

1. 查看 🔧问题排查与诊断指南.txt 找类似问题
2. 在 backend/main.py 或 frontend/index.html 中定位问题代码
3. 修复问题
4. 更新相关文档
5. 运行 quick_test.py 验证修复
6. 发布新版本

【优化性能的步骤】

1. 查看 完整系统验证报告.md 中的性能指标
2. 使用 Python profiler 找瓶颈
3. 优化代码（通常是数据库查询或 AI 调用）
4. 测试新性能，更新报告
5. 发布更新
