## 🔧 我想发布应用，应该做什么？

### 第1步：生成发布包
在项目根目录运行：
```bash
python release_all.py
```

等待3-5分钟，生成文件到 `releases/` 目录：
- `AI-Study-Tool-v1.0-source-20260608.zip` (源码版)
- `AI-Study-Tool-v1.0-standalone-20260608-win64.zip` (一键版)
- `README.md` (发布说明)

### 第2步：上传到GitHub Releases
1. 登录GitHub
2. 进入项目仓库
3. 点击 "Releases" 标签
4. 点 "Create new release"
5. 上传刚生成的3个文件

### 第3步：分享下载链接
```
给小白用户: 下载 standalone-win64.zip，按照 小白使用说明.md 操作
给开发者: 下载 source.zip，按照 INSTALLATION.md 操作
```

## 🚀 常见任务速查

### 我想运行系统
```bash
# Windows
双击 backend/start.bat

# Mac/Linux
bash backend/start.sh
```

### 我想修改配置
编辑 `backend/config.json`，修改：
- `port` - 服务端口
- `api_key` - DeepSeek API Key
- `review_days` - 复习间隔
- `crawl_sources` - 爬虫源

### 我想发布新版本
```bash
python release_all.py
```

### 我想测试系统
```bash
cd backend
python quick_test.py
```

### 我想查看API文档
启动后端，访问：
`http://127.0.0.1:8000/docs`

### 我想清空数据
删除 `backend/app.db` 文件，重启系统自动重建
