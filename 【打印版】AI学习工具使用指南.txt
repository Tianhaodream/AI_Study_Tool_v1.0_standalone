【第 1 页：快速启动】
═══════════════════════════════════════════════════════

🎯 只需三步：

    [第 1 步] 找到文件夹 → AI_Study_Tool

    [第 2 步] 在文件夹里找 → 启动系统.bat
              用鼠标 → 双击它！

    [第 3 步] 等待 3-5 秒
              黑窗口闪一下（正常！）
              浏览器自动打开

✨ 如果浏览器没打开？
    → 等 10 秒后，手动访问：http://localhost:8000
    → 看到程序界面就说明成功了！


【第 2 页：配置 API 密钥（只需一次）】
═══════════════════════════════════════════════════════

🔑 为什么需要？
    → 让程序能访问 AI（ChatGPT）来帮你分析文章

📝 怎样获取（3 个步骤）：

    ① 打开浏览器，访问：
       https://platform.openai.com/api/keys
    
    ② 注册账号（没有？用 Google 或 Microsoft 账号登录）
    
    ③ 点"Create new secret key"
       → 复制密钥（注意不要多复制空格！）
       → 粘贴到程序里

💾 在程序里配置：
    
    ① 点右上角 ⚙️ 设置按钮
    
    ② 找"API 密钥"输入框
    
    ③ 粘贴你的密钥 → 点保存
    
    ④ 刷新页面（按 F5）
    
    ✅ 完成！以后不用再配了


【第 3 页：开始使用（4 大功能）】
═══════════════════════════════════════════════════════

📚 功能 1：【抓取文章】
    
    怎么用？
    • 在左侧输入框粘贴一篇文章、新闻、或任何文本
    • 点"抓取"按钮
    • 自动保存！
    
    什么时候用？
    • 阅读到好内容时，想保存下来复习


✏️  功能 2：【标注笔记】
    
    怎么用？
    • 找到你抓取的文章
    • 双击某个句子
    • 在弹出框输入你的笔记、想法、评论
    • 点"保存"
    
    什么时候用？
    • 标记重点、写感受、记理解


🧠 功能 3：【AI 分析】
    
    怎么用？
    • 找到某篇文章
    • 点"AI 分析"按钮
    • 等待 10-30 秒（AI 在思考，不要急！）
    • 看到 AI 生成的摘要、关键词、学习建议
    
    什么时候用？
    • 想快速理解文章内容
    • 需要学习建议
    • 想看 AI 的分析观点


📖 功能 4：【历史记录】
    
    怎么用？
    • 点左侧"历史记录"
    • 看到所有你保存过的文章和笔记
    • 点任何一条可以继续编辑
    
    什么时候用？
    • 想回顾以前的学习内容
    • 想找某篇文章
    • 想修改笔记


【第 4 页：常见问题速查】
═══════════════════════════════════════════════════════

❓ 问：启动后什么都没有？
✅ 答：
   • 等 5 秒，浏览器还没反应？
   • 打开浏览器 → 在地址栏输入 http://localhost:8000
   • 还是不行？关闭黑窗口，重启一次

❓ 问：看到红色警告"API 密钥无效"？
✅ 答：
   • 检查你的密钥是否正确复制（有没有多复制空格？）
   • 确保 OpenAI 账户有余额（免费账户可能需要充值）
   • 在"设置"菜单重新输入一次

❓ 问：AI 分析很慢，转圈圈转了 1 分钟？
✅ 答：
   • 正常！AI 处理确实需要 5-30 秒
   • 不要关闭窗口
   • 耐心等待就行

❓ 问：数据保存后，刷新页面就丢了？
✅ 答：
   • 数据没丢！刷新一下页面（按 F5）
   • 在"历史记录"里找一下
   • 重启程序也能看到

❓ 问：复制粘贴文本时出错？
✅ 答：
   • 不要一次性复制太长的文本（超过 100 页）
   • 分次粘贴（分成 5-10 页为宜）
   • 用右键菜单的"粘贴"试试

❓ 问：忘记配置 API 密钥了？
✅ 答：
   • 没关系，随时可以配
   • 点右上角"设置" → 输入 API 密钥 → 保存


【第 5 页：数据和安全】
═══════════════════════════════════════════════════════

🔒 我的数据在哪里？
   
   → 全部保存在你的电脑上（app.db 文件）
   → 不上传到云端
   → 100% 安全


📁 我可以备份我的数据吗？

   → 可以！直接复制 app.db 文件就行

💡 我能离线使用吗？

   → 可以！本地的"抓取"和"标注"不需要网络
   → "AI 分析"需要网络（要调用 OpenAI 的 AI）
