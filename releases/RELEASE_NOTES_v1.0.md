## 🚀 快速开始

### 三步启动

#### 1️⃣ 解压文件
```bash
# Windows用户右键解压
# Mac/Linux用户
unzip AI-Study-Tool-v1.0.zip
```

#### 2️⃣ 进入backend目录并运行脚本
```bash
# Windows
cd backend
install.bat

# Mac/Linux
cd backend
bash start.sh
```

#### 3️⃣ 配置API Key
编辑 `backend/config.json`，填入你的DeepSeek API Key：
```json
"api_key": "sk-你的Key"
```

重新运行脚本启动即可使用！

### 详细步骤

请参考 **INSTALLATION.md** （完整安装指南）

## 💡 使用流程示例

```
1. 点击[获取今日文章]
   → 系统自动爬取一篇人民网评论

2. 阅读文章，选中关键文本
   → 点[标记问题/原因/对策] → 添加颜色标注和笔记

3. 完成标注后，点[AI深度分析]
   → AI返回结构化分析结果（10-20秒）

4. 查看分析结果，对比自己的理解
   → 系统生成复习计划（1,2,4,7,15天）

5. 下次登陆，点历史记录复习该文章
   → 重复直至掌握
```
