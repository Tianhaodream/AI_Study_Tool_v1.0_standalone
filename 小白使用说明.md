# 🎓 小白专用使用说明

**给电脑小白的完整指南** - 一步步教你使用AI评论深度训练系统

---

## 什么是这个系统？

这是一个**智能学习助手**，帮助你：
- 📰 自动获取最新的新闻评论
- 🏷️ 标记重要信息
- 🤖 用AI分析文章内容
- 📚 制定复习计划

**完全免费使用**（只需一个免费的API Key）

---

## ⚡ 5分钟快速开始

### 第1步：下载软件

1. 打开浏览器，访问：[项目Releases页面](../../releases)
2. 找到 **`AI-Study-Tool-v1.0-standalone-***-win64.zip`** 
3. 点击下载（下载完后是一个zip文件）

### 第2步：解压文件

1. 找到下载好的 `.zip` 文件
2. **右键点击** → 选择 **"解压到此处"**
3. 等待解压完成（会出现一个文件夹）

### 第3步：启动程序

1. 进入解压后的文件夹
2. 找到文件 **`start.bat`**（图标是一个黑色的方块）
3. **双击它**
4. 稍等3-5秒，程序自动启动
5. 浏览器会自动打开，显示系统界面

### 第4步：获取API Key（首次必须）

首次启动时，程序会自动打开一个 **`config.json`** 的文件（记事本形式）

1. **打开浏览器**，访问：https://platform.deepseek.com/
2. 点击 **"开始使用"** 或 **"登录"**
3. 用邮箱或手机注册账户
4. 登录后，找到 **"API管理"** 或 **"API Keys"**
5. 点击 **"创建API Key"**
6. 复制生成的Key（看起来像：`sk-xxxxx...`）
7. 回到 **config.json** 文件，找到这一行：
   ```json
   "api_key": ""
   ```
8. 把引号中间填入你复制的Key：
   ```json
   "api_key": "sk-你复制的内容"
   ```
9. **Ctrl+S** 保存，然后关闭文件
10. **重新启动** `start.bat`

✅ 完成！程序现在可以使用了

## 📖 基本操作

### 界面说明

```
┌─────────────────────────────────┐
│   AI 评论深度训练系统           │
├─────────────────────────────────┤
│                                 │
│  [获取今日文章] [AI深度分析]    │  ← 这些是按钮

│  🔴 标记问题 🟡 标记原因 🟢 标记对策
│                                 │
├─────────────────────────────────┤
│  历史练习记录                   │
│  - 文章1                         │
│  - 文章2                         │
│  ...                             │
└─────────────────────────────────┘
```

### 4个基本功能

#### ① 获取今日文章 📰
1. 点击 **[获取今日文章]** 按钮
2. 等待3-5秒
3. 一篇新的评论文章就加载出来了
4. 仔细阅读文章内容

#### ② 标注重点 🏷️
阅读时遇到重要内容，按照以下步骤标记：

**标记问题**（红色）
- 选中文章中说的 **问题** 部分
- 点击 **[标记问题]** 按钮
- 可选：输入个人笔记（比如"很重要"）
- 点击保存

**标记原因**（黄色）
- 选中造成问题的 **原因** 部分
- 点击 **[标记原因]** 按钮

**标记对策**（绿色）
- 选中解决方案或 **对策** 部分
- 点击 **[标记对策]** 按钮

**如何查看笔记**：
- 把鼠标放在有颜色的文字上，会显示你的笔记

#### ③ AI深度分析 🤖
1. 点击 **[AI深度分析]** 按钮
2. 等待10-20秒（第一次会比较慢）
3. AI会给出三个分析结果：
   - 🚩 **核心问题** - 文章讲的什么问题
   - 🔍 **深度原因** - 为什么会有这个问题
   - 💡 **应对对策** - 怎样解决这个问题
4. 结果会自动保存

#### ④ 查看历史 📚
下方的 **"历史练习记录"** 显示你之前学过的所有文章：

- 点击文章名字可以重新查看
- 点击旁边的 🗑️ 图标可以删除

## 💡 使用建议

### 每天学习流程

1. **早上**：打开程序，点击"获取今日文章"
2. **阅读**：仔细读文章，把重要内容标记出来（5-10分钟）
3. **分析**：点击"AI深度分析"，看AI的分析结果（2-3分钟）
4. **对比**：把自己的理解和AI的分析进行对比，找差距

### 复习方法

系统会自动在以下时间提醒你复习：
- 1天后
- 2天后
- 4天后
- 7天后
- 15天后

每次复习时：
1. 从"历史练习记录"中找出那篇文章
2. 点开查看你之前的标注
3. 看看有没有遗漏的地方
4. 加深印象

### 坚持要点

- 📝 **每天至少一篇** - 质量比数量重要
- 🎯 **认真标注** - 标注越仔细，学习效果越好
- 🔄 **定期复习** - 跟着系统提示的日期复习
- 💭 **思考对比** - 对比自己和AI的分析，提高理解能力

## ❓ 常见问题

### Q1: 程序启动时出现黑窗口就消失了
**原因**：通常是系统检测到问题
**解决**：
1. 打开文件夹，找到 `config.json`
2. 用记事本打开，检查内容是否有错误
3. 特别查看 `"api_key"` 后面有没有正确粘贴Key

### Q2: 浏览器打开了但显示"无法连接"
**原因**：后端服务没有正常启动
**解决**：
1. 找那个黑窗口（后端窗口）
2. 看看有没有错误信息
3. 尝试关闭黑窗口，重新双击 `start.bat`
4. 等5秒后再刷新浏览器

### Q3: "获取今日文章"点了没反应
**原因**：可能是网络问题
**解决**：
1. 检查电脑网络是否正常
2. 等一会再试
3. 或者检查是否被学校/公司防火墙阻止

### Q4: 点"AI深度分析"显示错误
**原因**：最常见的是API Key没有正确填入
**解决**：
1. 打开 `config.json`
2. 确认 `"api_key"` 后面的值是正确的
3. 确保前后都有双引号：`"api_key": "sk-xxxxx"`
4. 保存，重启程序

### Q5: 找不到 config.json 在哪
**解决**：
1. 进入解压后的文件夹
2. 再进入 **`backend`** 文件夹
3. 在那里就能看到 `config.json` 了

## 📱 使用技巧

### 技巧1：复制粘贴
- 选中文字时，可以 **Ctrl+C 复制** 
- 粘贴时用 **Ctrl+V**
- 这样可以快速标注重点

### 技巧2：快速查找
- 按 **Ctrl+F** 可以在文章中搜索关键词
- 比如搜索"问题"，就能快速定位

### 技巧3：备份数据
- 程序自动保存所有数据
- 数据保存在 **`app.db`** 文件中
- 如果想备份，复制整个文件夹就可以了

### 技巧4：多台电脑使用
- 一个 API Key 可以在多台电脑上使用
- 只需要在每台电脑的 `config.json` 中填入同一个Key就行

## 🚀 进阶功能（可选）

### 调整复习间隔
如果你觉得复习时间太长或太短，可以修改：

打开 `config.json`，找到这一行：
```json
"review_days": [1, 2, 4, 7, 15]
```

改为你喜欢的间隔，比如：
```json
"review_days": [1, 3, 7]
```

保存后重启程序。

### 添加更多爬虫源
你可以添加其他网站的内容。打开 `config.json`，找到：
```json
"crawl_sources": [
    "http://opinion.people.com.cn/GB/436867/index.html",
    "http://opinion.people.com.cn/GB/8213/49160/49219/index.html"
]
```

可以添加其他评论网站的链接。
