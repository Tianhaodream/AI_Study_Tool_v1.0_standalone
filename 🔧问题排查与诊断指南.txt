═【场景 1】启动时一片黑（什么都没有发生）════════════════════════════════

症状：双击"启动系统.bat"后，黑窗口出现了但网页没打开

解决步骤：
  ✓ 第 1 步：等待 5 秒钟（程序在后台启动）
  ✓ 第 2 步：打开浏览器，手动访问：http://localhost:8000
           (可以是 Chrome、Edge、Firefox 任何浏览器)
  ✓ 第 3 步：如果出现程序界面，说明启动成功！
  ✓ 第 4 步：如果还是不行，参考【场景 3】
  ✓ 第 5 步：实在不行，看文档末尾的【高级诊断】

时间长度：通常需要 3-5 秒，最多不超过 10 秒


═【场景 2】浏览器打开了，但显示"无法连接"════════════════════════════════

症状：浏览器显示"ERR_CONNECTION_REFUSED" 或 "无法到达此网站"

原因：后端服务没有正常启动

解决步骤：
  ✓ 第 1 步：关闭所有浏览器窗口
  ✓ 第 2 步：找到那个黑窗口，关闭它（点右上角 X）
  ✓ 第 3 步：重新双击"启动系统.bat"
  ✓ 第 4 步：等待 10 秒钟，再访问 http://localhost:8000
  ✓ 第 5 步：还是不行？看【高级诊断】


═【场景 3】程序打开了，但显示"请设置 API 密钥"════════════════════════════

症状：页面顶部显示红色警告："❌ 请配置 OpenAI API 密钥"

解决步骤：
  ✓ 第 1 步：在浏览器右上角找"⚙️ 设置"按钮，点击
  ✓ 第 2 步：在弹出框里找"API 密钥"输入框
  ✓ 第 3 步：去 https://platform.openai.com/api/keys 获取密钥
           (需要注册 OpenAI 账号，可以用 Google 或 Microsoft 账号)
  ✓ 第 4 步：复制密钥（注意不要多复制空格！）
  ✓ 第 5 步：粘贴到输入框，点"保存"
  ✓ 第 6 步：刷新页面（按键盘的 F5），应该就没有警告了

时间长度：5-10 分钟（主要是注册账号的时间）
费用：OpenAI API 第一次注册通常有免费额度，用完后按使用量付费


═【场景 4】"AI 分析"功能很慢或无反应════════════════════════════════════

症状：点"AI 分析"后，转圈圈转了很久（1分钟以上）

这是正常现象！AI 处理确实需要时间。

解决步骤：
  ✓ 第 1 步：不要关闭浏览器窗口，耐心等待
  ✓ 第 2 步：如果等了 2 分钟还没有结果，刷新页面（F5）
  ✓ 第 3 步：重新尝试点"AI 分析"
  ✓ 第 4 步：还是不行？
           - 检查网络连接（试试能不能打开百度首页）
           - 查看 OpenAI 账户是否有余额
           - 看【高级诊断】部分

时间长度：正常情况下 5-30 秒就能出结果


═【场景 5】保存数据后，刷新页面数据就丢了════════════════════════════════

症状：标注的笔记、AI 分析结果在刷新后消失了

解决步骤：
  ✓ 第 1 步：刷新页面（F5）
  ✓ 第 2 步：在左侧"历史记录"里找一下，数据可能还在
  ✓ 第 3 步：重启程序：
           - 关闭浏览器
           - 关闭黑窗口
           - 重新双击"启动系统.bat"
  ✓ 第 4 步：检查是否网络连接中断了（数据可能没保存到后端）
  ✓ 第 5 步：看【高级诊断】


═【场景 6】文件上传或复制粘贴时出错═══════════════════════════════════════

症状：复制文本时出现错误，或无法粘贴长文本

解决步骤：
  ✓ 第 1 步：不要一次性复制太长的文本（超过 100 页最好分次）
  ✓ 第 2 步：试试用右键菜单的"粘贴"，而不是 Ctrl+V
  ✓ 第 3 步：如果还是不行，分批粘贴：
           - 分成多个较短的文本（5-10 页为宜）
           - 分别粘贴和保存


═【场景 7】看到乱码或奇怪的符号═══════════════════════════════════════════

症状：中文显示成了"??"或奇怪的方块

这通常是编码问题。

解决步骤：
  ✓ 第 1 步：刷新浏览器（Ctrl+Shift+Delete 清除缓存，然后 F5）
  ✓ 第 2 步：换个浏览器试试（比如用 Edge 代替 Chrome）
  ✓ 第 3 步：重启程序


═════════════════════════════════════════════════════════════════════════
📋 高级诊断（给有点基础的用户）
═════════════════════════════════════════════════════════════════════════

如果上面的步骤都试过了还是不行，试试这个：

【打开命令行调试】
  1. 按 Windows 键，输入"cmd"，按 Enter
  2. 在黑窗口里输入：
     cd D:\AI_Study_Tool
  3. 再输入：
     python backend/main.py
  4. 程序会显示详细的错误信息
  5. 看看是否有红色的"ERROR"或"Exception"字样
  6. 把错误信息截图，发给技术支持

【检查文件完整性】
  1. 在 AI_Study_Tool 文件夹里看一下：
     • 有没有 backend/ 文件夹（里面应该有 main.py）
     • 有没有 frontend/ 文件夹（里面应该有 index.html）
     • 有没有 config.json 或 config_template.json 文件
  2. 如果缺少这些文件，说明下载的文件包不完整，需要重新下载

【检查磁盘空间】
  1. 右键点击 C: 盘（或其他安装盘），选"属性"
  2. 检查是否还有至少 500MB 的空闲空间
  3. 如果空间不足，删除一些不需要的文件
