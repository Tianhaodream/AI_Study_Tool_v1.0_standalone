╔════════════════════════════════════════════════════╗
║   AI 评论深度训练系统 - 极速上手卡                ║
║                  (可打印版)                        ║
╚════════════════════════════════════════════════════╝

┌─ 第1步：下载和解压 ──────────────────────────────┐
│                                                   │
│  📥 下载：releases 页面下载 standalone-win64.zip  │
│  📂 解压：右键 → 解压到此处                       │
│                                                   │
└───────────────────────────────────────────────────┘

┌─ 第2步：获取 API Key ────────────────────────────┐
│                                                   │
│  🌐 打开：https://platform.deepseek.com/         │
│  📝 注册：用邮箱注册                              │
│  🔑 创建：在API管理中创建新的API Key              │
│  📋 复制：复制 sk-开头的长字符串                  │
│                                                   │
└───────────────────────────────────────────────────┘

┌─ 第3步：填入 API Key ────────────────────────────┐
│                                                   │
│  📄 打开：backend 文件夹中的 config.json         │
│  ✏️  编辑：找到这一行：                           │
│     "api_key": ""                                │
│  🔧 替换：粘贴 Key：                              │
│     "api_key": "sk-你的Key"                      │
│  💾 保存：Ctrl+S                                 │
│                                                   │
└───────────────────────────────────────────────────┘

┌─ 第4步：启动程序 ────────────────────────────────┐
│                                                   │
│  🎬 双击：start.bat                              │
│  ⏳ 等待：3-5 秒                                  │
│  🌐 打开：浏览器自动显示                          │
│                                                   │
└───────────────────────────────────────────────────┘

╔════════════════════════════════════════════════════╗
║             4 个基本操作                          ║
╠════════════════════════════════════════════════════╣
║                                                   ║
║ 1️⃣  获取今日文章                                  ║
║    点击按钮 → 等待 → 阅读文章                   ║
║                                                   ║
║ 2️⃣  标注重点                                      ║
║    选文字 → 点"标记问题/原因/对策"               ║
║    可选：输入个人笔记                             ║
║                                                   ║
║ 3️⃣  AI 分析                                       ║
║    点 AI深度分析 → 等待10秒 → 查看结果            ║
║    AI 给出：核心问题 + 原因 + 对策                ║
║                                                   ║
║ 4️⃣  查看历史                                      ║
║    点下方历史记录中的文章 → 重新查看               ║
║                                                   ║
╚════════════════════════════════════════════════════╝

┌─ 快速故障排查 ────────────────────────────────────┐
│                                                   │
│ ❌ start.bat 打开后闪退                           │
│    → 检查 config.json 中的 api_key               │
│    → 确保没有拼写错误                             │
│                                                   │
│ ❌ 浏览器显示"无法连接"                            │
│    → 重启 start.bat                              │
│    → 等待 5 秒后刷新浏览器                         │
│    → 看看有没有黑窗口（后端服务）                 │
│                                                   │
│ ❌ AI 分析出错                                    │
│    → 检查 API Key 是否正确                        │
│    → 检查网络是否正常                             │
│    → 检查 DeepSeek 账户是否有余额                 │
│                                                   │
└───────────────────────────────────────────────────┘
