# 📝 更新日志 (CHANGELOG)

所有值得注意的项目变更都会记录在此文件中。

## [1.0.0] - 2026-06-06

### 初始发布 🎉

#### ✨ 新增功能

**核心功能**：
- 🌐 从人民网自动爬取评论文章
- 🤖 AI深度分析（基于DeepSeek API）
- 🎨 文章标注系统（问题/原因/对策三色标注）
- 💾 个人笔记和批注
- 📅 智能复习计划（基于遗忘曲线）
- 📚 学习历史记录管理

**用户界面**：
- 现代化Web界面（Tailwind CSS）
- 响应式设计，适配各种屏幕
- 直观的标注和分析工具
- 实时预览和交互

**后端服务**：
- FastAPI高性能框架
- SQLite本地数据库
- RESTful API设计
- CORS跨域支持

#### 🛠️ 改进

**安装体验**：
- ✅ 一键安装脚本（Windows/Mac/Linux）
- ✅ 自动虚拟环境创建
- ✅ 自动依赖安装
- ✅ 自动配置初始化

**配置管理**：
- ✅ JSON配置文件系统
- ✅ 环境变量支持
- ✅ 敏感信息保护（.gitignore）
- ✅ 灵活的可配置参数

**API Key管理**：
- ✅ 前端localStorage自动保存
- ✅ 页面刷新自动加载
- ✅ 用户可手动清除
- ✅ 安全的本地存储

**文档**：
- ✅ README.md - 完整项目文档（6000+字）
- ✅ QUICK_START.md - 5分钟快速开始
- ✅ INSTALLATION.md - 详细安装指南（6700+字）
- ✅ INSTALL_QUICK_REFERENCE.md - 快速参考卡
- ✅ CHANGELOG.md - 更新日志

**打包分发**：
- ✅ setup.py Python打包配置
- ✅ 自动打包脚本（pack.py）
- ✅ 轻量化zip发布包（~25KB）
- ✅ releases目录和版本管理

#### 🔐 安全性

- API Key从代码库中分离
- config.json已加入.gitignore
- 支持环境变量配置
- 数据仅存储本地，不上传服务器

#### 📊 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端 | HTML5 + Tailwind CSS | 3.3.6 |
| 后端 | FastAPI | 0.104.1 |
| 服务器 | Uvicorn | 0.24.0 |
| 爬虫 | BeautifulSoup4 | 4.12.2 |
| 数据库 | SQLite | 3.x |
| 配置 | Pydantic | 2.5.0 |

#### 🎯 系统要求

- Python 3.8+
- Windows 7+ / Mac OS 10.14+ / Linux (Ubuntu 18.04+)
- 100MB磁盘空间
- 网络连接（可选，本地功能可离线使用）

#### 📦 发布包

- **轻量化包**：~25KB（推荐分发）
  - 包含所有源代码
  - 用户自行安装Python依赖
  - 跨平台兼容

- **独立可执行包**：（计划v1.1）
  - PyInstaller打包
  - 无需Python环境
  - 专供Windows用户

#### 📋 已知限制

- 仅支持人民网评论栏目爬虫
- 需要有效的DeepSeek API Key
- 数据存储在本地，多设备需手动同步
- 前端localStorage存储有浏览器大小限制

#### 🐛 已知问题

- 暂无重大bug

#### 🙏 致谢

感谢所有测试和反馈的用户！
