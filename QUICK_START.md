# ⚡ 快速开始（5分钟入门）

## 🎯 30秒概览

**AI评论训练系统** 是一个智能学习助手，帮助你通过AI分析深入学习新闻评论。

核心流程：
1. 获取文章 → 2. 标注关键内容 → 3. AI深度分析 → 4. 复习巩固

## ⚡ 最快开始（推荐！一键启动）

### Windows 用户

1. 下载 `AI-Study-Tool-v1.0-standalone-*.zip`（从 Releases 页面）
2. 解压到任意位置
3. **双击 `start.bat` 启动系统**
4. 浏览器自动打开，按提示填入API Key即可

✅ **不需要安装Python**

### Mac / Linux 用户

1. 下载对应版本 `.zip`
2. 解压后运行：`bash start.sh`
3. 浏览器自动打开，按提示填入API Key即可

✅ **不需要安装Python**

---

## 🔑 获取API Key（一次性配置）

首次运行后，系统会自动创建 `config.json` 文件。

### 第1步：获取DeepSeek API Key
1. 访问 [DeepSeek官网](https://platform.deepseek.com)
2. 注册账户（推荐用邮箱注册）
3. 创建 API Key
4. 复制你的 Key（格式：`sk-xxxxxxx...`）

### 第2步：填入配置
编辑 `config.json` 文件，找到这一行：
```json
"api_key": "",
```

替换为你的真实 API Key：
```json
"api_key": "sk-你的Key",
```

保存文件，重启系统即可。

## 💡 功能使用教程

### 1️⃣ 获取今日文章
点击 **[获取今日文章]** 按钮
- 系统从人民网随机获取一篇评论文章
- 自动提取正文内容（已过滤广告）
- 生成智能复习计划（1,2,4,7,15天）

### 2️⃣ 标注关键内容
阅读文章，选中重要文本，点击：
- 🔴 **标记问题** - 标记文章描述的问题
- 🟡 **标记原因** - 标记导致问题的原因  
- 🟢 **标记对策** - 标记解决问题的方案

可选：输入个人批注（悬停显示）

### 3️⃣ AI深度分析
点击 **[AI深度分析]** 按钮（需要API Key）
- AI分析文章中的核心问题
- 深度挖掘原因
- 提出应对对策
- 结果自动保存

### 4️⃣ 复习学习
点击下方 **📂 历史练习记录** 中的文章
- 快速回顾之前的标注和笔记
- 查看AI分析结果
- 重复学习直至掌握

## 🛠️ 开发者安装（源码版）

如需修改代码或参与开发：

### 要求
- Python 3.8+
- 命令行工具

### 安装步骤

```bash
# 1️⃣ 进入backend文件夹
cd backend

# 2️⃣ 安装依赖
pip install -r requirements.txt

# 3️⃣ 启动系统

```

## 🐛 常见问题

### Q1: 启动后黑窗口闪退
**解决**：
- 检查 `config.json` 是否存在
- 查看是否有Python错误信息
- 尝试手动运行：在该目录打开命令行，输入 `ai-study-tool-backend.exe` 查看错误

### Q2: 前端无法连接后端
**解决**：
1. 确认后端运行（是否还有黑色窗口）
2. 检查后端地址：`http://127.0.0.1:8000`
3. 检查防火墙是否阻止

### Q3: AI分析提示API Key错误
**解决**：
1. 检查 `config.json` 中的 api_key 是否正确
2. 检查DeepSeek账户是否有余额
3. 检查Key是否过期（重新生成新的）

### Q4: 数据库锁定错误
**解决**：
1. 重启后端：关闭黑窗口，重新运行脚本
2. 如果还不行，删除 `app.db` 文件（会重建）
