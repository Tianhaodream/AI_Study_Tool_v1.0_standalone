🎯 三步开始使用（傻瓜级操作）
═══════════════════════════════════════════════════════════════════════════

【第 1 步】双击启动程序
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📂 找到项目文件夹
   └─ 👉 找到一个叫"AI_Study_Tool"的文件夹（可能在桌面或下载文件夹）

⚙️  运行启动文件
   └─ 👉 在文件夹里找到"启动系统.bat"，用鼠标双击它
   └─ 👉 会看到一个黑色窗口快速闪过，这是正常的
   └─ 👉 然后自动打开浏览器，显示程序界面

⏳ 如果什么都没打开
   └─ 👉 等待 3-5 秒后刷新浏览器（按 F5）
   └─ 👉 如果还是不行，看下面【常见问题】


【第 2 步】设置 API 密钥（重要！只需一次）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第一次启动时，程序会提示您输入一个 API 密钥（用来访问 AI 功能）。

🔑 获取 API 密钥（免费）：
   👉 打开浏览器，访问：https://platform.openai.com/api/keys
   👉 注册一个 OpenAI 账号（或用微软/Google账号登录）
   👉 创建新的 API 密钥，复制保存下来
   👉 回到程序，粘贴到"API 密钥"输入框
   👉 点击"保存"按钮

💡 提示：
   • 第一次配置后就不需要再输入了，程序会记住
   • 如果 API 配置错误，会在页面右上角显示警告
   • 重新配置：在"设置"菜单里更新 API 密钥即可


【第 3 步】开始学习！
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

现在你可以使用所有功能了：

📚 【抓取文章】
   1. 在左侧输入框粘贴一篇文章、新闻或任何文本
   2. 点击"抓取"按钮
   3. 程序自动保存文章

✏️  【标注重点】
   1. 找到你抓取的文章
   2. 双击任意句子，会弹出编辑框
   3. 输入你的笔记、理解或评论
   4. 点击"保存"

🧠 【AI 分析】
   1. 对着某篇文章，点击"AI 分析"
   2. 程序自动用 AI 生成总结、关键词、学习建议
   3. 保存结果供日后复习

📖 【历史记录】
   • 所有操作都自动保存
   • 点击左侧"历史记录"查看过去的文章和笔记
   • 随时可以继续编辑和补充

⚠️  常见问题（99% 的问题都在这儿）
═══════════════════════════════════════════════════════════════════════════

❓ 问：双击"启动系统.bat"后什么都没有？
✅ 答：
   • 等待 3-5 秒，然后手动打开浏览器访问 http://localhost:8000
   • 如果还是不行，打开黑色窗口（cmd），输入命令查看错误信息
   • 查看下方【技术支持】部分

❓ 问：出现"API 密钥无效"的警告？
✅ 答：
   • 你的 API 密钥配置有误
   • 重新检查是否正确复制了密钥（不要有多余空格）
   • 确保 OpenAI 账户有余额（免费账户可能需要充值）
   • 在"设置"菜单更新 API 密钥

❓ 问：文章无法保存？
✅ 答：
   • 刷新页面（按 F5）重试
   • 检查你的网络连接是否正常
   • 如果还是不行，重启程序（关闭黑色窗口后再启动一次）

❓ 问：AI 生成摘要时很慢？
✅ 答：
   • 这是正常的，AI 处理通常需要 5-30 秒
   • 不要关闭窗口，耐心等待
   • 检查你的网络连接质量

❓ 问：程序启动失败或闪退？
✅ 答：
   • 打开 cmd 命令行，进入项目文件夹
   • 输入命令查看详细错误：
     cd D:\AI_Study_Tool
     python backend/main.py
   • 把错误信息记下来，参考【技术支持】部分
